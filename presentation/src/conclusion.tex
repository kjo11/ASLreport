\section{Conclusion}

\frame{\frametitle{Achievements}

\begin{itemize}
  \item Created versatile \texttt{stereo surface reconstruction} package for
    \begin{itemize}
      \item variable spline degree and resolution,
      \item entirely customizable optimization parameters and
      \item rectified and unrectified images.
    \end{itemize}
  \item Implemented photometric localization algorithm based on one stereo
    measurement.
  \item Tested functionalities on real and simulated datasets.
\end{itemize}
